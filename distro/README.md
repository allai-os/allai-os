# Distro build

Construcción de la imagen de allAI OS: RPMs, imagen OCI rpm-ostree, ISO instalable, branding, kickstart.

**Estado**: estructura pendiente. Ver [ROADMAP.md](../ROADMAP.md) fase A — Assemble.

## Subcarpetas previstas

- `rpms/` — SPEC files de paquetes propios
- `ostree/` — Containerfile y composiciones rpm-ostree
- `kickstart/` — archivos `.ks` para Anaconda
- `branding/` — logo, wallpapers, plymouth, gdm theme
- `ci/` — workflows de build y firma
